这是remix icon@3.2.0版本的微信小程序实现。



使用小程序开发工具的终端输入

```
npm i remixicon-weapp`
```



完成后使用“工具 - 构建npm”



app.wsxx引用：

```
@import './miniprogram_npm/remixicon-weapp/remixicon.wxss';
```
